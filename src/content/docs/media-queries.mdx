## Available Media Queries

**Important:** [custom media queries are](https://drafts.csswg.org/mediaqueries-5/#custom-mq) are not yet available without a [PostCSS plugins](https://github.com/csstools/postcss-plugins/tree/main/plugins/postcss-custom-media). If you can't or don't want to add a build step to your process, you'll have to hard code your breakpoints. (You can still use nesting as in the example below though.)

### Dimensions

| Token keyword | Dimension (px) |
| --------------|----------------|
| `xxs`         | 0-240          |
| `xs`          | 240-360        |
| `sm`          | 360-480        |
| `md`          | 480-768        |
| `lg`          | 768-1024       |
| `xl`          | 1024-1440      |
| `xxl`         | 1440-1920      |

### Variations

Using the `md` dimension as an example:

| Token variation | Matches |
| ----------------|---------|
| `--md-only` | exact range |
| `--md-n-above` | range top and above |
| `--md` | shorthand for `--md-n-above` |
| `--md-n-below` | range top and below |
| `--md-phone` | exact range in portrait only |

### Examples

| Custom media query | Value |
|--------------------|------|
| `@media(--md-only) {}`    |  `(480px <= width < 768px);` |
| `@media(--md) {}` |  `(width >= 768px);` |
| `@media(--md-n-above) {}` |  `(width >= 768px);` |
| `@media(--md-n-below) {}` |  `(width < 768px);` |
| `@media(--md-phone) {}`   |  `(--md-only) and (--portrait);` |


## Recommended responsive setup

If you search online for the best appraoch to responsive design and setting up your breakpoints, you'll come accross the technically true but useless "it depends" answer.

Unless you have a good reason not to, you should use a mobile first approach. What that means is you design will work great on small screen out of the box, without any media queries, and then you add your tweaks for additional sizes.

In short, here's an (silly) example of how the CSS of  most responsive components should be set up:

```html
<style>
.exampleComponent {
    color: var(--base-50);
    background-color: var(--base-900);
    @media(--lg) {
      color: var(--base-950);
      background-color: var(--base-200);

    }
}
</style>

</div>
```
