The grid layout is extremely easy to use, light weight, and very flexible. It uses the [CSS grid property](https://developer.mozilla.org/en-US/docs/Web/CSS/grid) under the hood, but it lets you set up your layouts directly into your HTML.

## Syntax

It's composed of 2 classes: `.l_grid` for the container, and `l_cell` for each grid item.

You define the amount of columns of your grid with a custom `col` attribute and the position/width of your cells with the custom `span` attribute.

The `span` property syntax follows these 5 patterns:

- `row`: the cell spans a full row
- `3`: the cell start in the next column and spans 3 columns.
- `3-5`: the cell starts in column 3 and ends in column 5.
- `3+5`: the cell starts in column 3 and spans for 5 columns.
- `3..`: the cell starts in column 3 and spans the remainder of rows.

Both the `col` and `span` attributes have responsive versions, which use the "mobile first" appraoch of the `--*-n-above` media queries.


## Examples

### Different types of spans

9 colum grid, with all types of spans

<div class="l_grid docs_grid" col="9">
  <div class="l_cell" span="2">2</div>
  <div class="l_cell" span="4-5">4-5</div>
  <div class="l_cell" span="6..">6..</div>
  <div class="l_cell" span="3-3">3-3</div>
  <div class="l_cell" span="5..">5..</div>
  <div class="l_cell" span="2+3">2+3</div>
  <div class="l_cell" span="6+2">6+2</div>
  <div class="l_cell" span="row">row</div>
  <div class="l_cell" span="4+2">4+2</div>
</div>

### Responsive grid

The grid layout follows a "mobile first" approach using `--*-n-above` media queries.

The following example goes from  1 → 4 → 9 → 12 columns.

```html
<div class="l_grid docs_grid" col="1" col-md="4" col-lg="9" col-xl="12">
  […]
</div>
```
