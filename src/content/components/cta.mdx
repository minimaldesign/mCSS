Two ways to ask for attention. The `.cta` component is the full-width closing section of a marketing page: big title, a line of text, buttons. The `.banner` component is the thin announcement bar at the very top of the page, optionally dismissible — the dismissal is remembered in `localStorage` and the one small inline script removes an already-dismissed banner before paint.

<Banner>
  <p>mCSS v1 is out! <a href="#">Read the announcement</a></p>
</Banner>

## HTML

<section class="prose">

```html
<section class="cta wrap cta-primary">
  <div class="cta_content">
    <h2 class="cta_title">Ready to own your CSS?</h2>
    <p class="cta_text">Copy the framework, edit the tokens, ship the site.</p>
    <div class="cta_actions">
      <a class="bt bt-lg bt-primary" href="/docs/start">Get started</a>
    </div>
  </div>
</section>
```

```html
<aside class="banner">
  <div class="banner_content">
    <p>mCSS v1 is out! <a href="/blog/v1">Read the announcement</a></p>
  </div>
  <button class="banner_close" aria-label="Dismiss announcement"><svg>[…]</svg></button>
</aside>
```

For a dismissible plain-HTML banner, copy the small inline script from the bottom of [Banner.astro][3]: it removes the banner when the stored key exists and stores the key on click.

### Available modifiers

<div class="docs_oversizedTable">

| Class                    | Description                                                     |
| ------------------------ | ---------------------------------------------------------------- |
| `.cta`                   | Default. Transparent background, centered content.               |
| `.cta` + `.cta-filled`   | Subtle filled band.                                              |
| `.cta` + `.cta-primary`  | Primary-colored band; text and links flip to the inverted colors.|

</div>

## Astro component

<div class="docs_oversizedTable prose">

**Cta**

| Prop      | Type     | Default     | Description                              |
| --------- | -------- | ----------- | ----------------------------------------- |
| `title`   | `string` | `undefined` | The big `h2`.                             |
| `text`    | `string` | `undefined` | Supporting line.                          |
| `variant` | `string` | `undefined` | `filled`, `primary`.                      |
| `class`   | `string` | `undefined` | Additional CSS classes.                   |

| Slot      | Description                                  |
| --------- | -------------------------------------------- |
| `actions` | The buttons row.                              |
| (default) | Extra content below the actions.              |

**Banner**

| Prop           | Type      | Default                   | Description                                             |
| -------------- | --------- | ------------------------- | -------------------------------------------------------- |
| `id`           | `string`  | `undefined`               | Required when dismissible; keys the `localStorage` entry (`mcss-banner-<id>`). Use a new id per campaign so a new announcement shows again. |
| `dismissible`  | `boolean` | `false`                   | Adds the close button + persistence script.              |
| `dismissLabel` | `string`  | `"Dismiss announcement"`  | Accessible name of the close button.                     |
| `class`        | `string`  | `undefined`               | Additional CSS classes.                                  |

The default slot is the banner content. Place the banner as the first child of `<body>`, above the header.

</div>
