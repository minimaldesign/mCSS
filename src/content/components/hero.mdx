The `Hero` component provides flexible hero layouts for landing pages, product sites, and marketing. ([Full size demo](/demos/hero).) It supports 3 variants:

- **Full:** full-bleed background with overlay/content.
- **Split:** side-by-side content and media.
- **Slideshow:** rotating slides with content/media.

<Hero
  variant="full"
  height={40}
  media={{ type: "canvas", src: "/_js/triangles.js" }}
  overlayOpacity={0.3}
  overlayColor="--primary-950"
  roundCorners
  textColor="--base-0"
  extraClass="mt-sm3"
>
  <h1>Example Hero</h1>
  <p>This hero uses an animated canvas element as background.</p>
</Hero>

#### Full layout (default)

The full layout spans the entire viewport with content (headline, subheadline, CTA) centered or aligned over a background image, video, gradient, or animated canvas. Use the `height` prop to control min-height in viewport units.

#### Split layout

The Split layout features an asymmetric split (e.g., `70/30`, `80/20`, etc.) with content on one side and media on the other. **Media fills its container, not a fixed aspect ratio.** For images, this means the image will cover the available space, cropping as needed.

#### Slideshow layout

The slideshow layout features rotating content (e.g., multiple headlines, CTAs, or background media) with pagination dots. Each slide can have its own media and content.

### Custom properties

The following custom properties are available in the [default theme](/docs/themes):

| Property                             | Description                                     |
| ------------------------------------ | ----------------------------------------------- |
| `--hero-radius`                      | Border radius (on/off via `roundCorners` prop). |
| `--hero-content-padding`             | Padding for hero content area.                  |
| `--hero-pagination-dot`              | Pagination dot size.                            |
| `--hero-pagination-dot-color`        | Pagination dot background color.                |
| `--hero-pagination-dot-active-color` | Active pagination dot color.                    |
| `--hero-pagination-opacity`          | Pagination dot default opacity.                 |

</section>

## Astro component

<section class="prose">

| Prop                  | Type    | Default   | Description                                                                   |
| --------------------- | ------- | --------- | ----------------------------------------------------------------------------- |
| `variant`             | string  | `full`    | Layout variant.                                                               |
| `ratio`               | string  | `62/38`   | Split/asymmetric ratio (must sum to 100, e.g. `70/30`, `40/60`, etc.).        |
| `contentFirst`        | boolean | `true`    | If `false`, content appears after media in DOM (content on right on desktop). |
| `direction`           | string  | `default` | Visually reverses the content order set by `contentFirst`.                    |
| `media`               | object  | —         | Image, video, gradient, or canvas.                                            |
| `parallax`            | boolean | `false`   | Parallax effect (full variant only).                                          |
| `height`              | number  | —         | Min-height in vh (0-100) for full variant.                                    |
| `overlayOpacity`      | number  | `0`       | Overlay opacity for full/slideshow.                                           |
| `overlayColor`        | string  | —         | Overlay color (CSS var or value).                                             |
| `roundCorners`        | boolean | `false`   | Hero media has rounded corners?                                               |
| `textColor`           | string  | —         | Text color (CSS var or value).                                                |
| `slideshowItems`      | array   | —         | Items for slideshow.                                                          |
| `slideshowAutoplay`   | boolean | `true`    | Slideshow autoplay.                                                           |
| `slideshowInterval`   | number  | `5000`    | Slideshow interval (ms).                                                      |
| `slideshowTransition` | number  | `600`     | Slideshow transition speed (ms).                                              |
| `extraClass`          | string  | —         | Extra CSS classes.                                                            |

</section>
