## Specs

<section class="prose">

| File Name           | class              | Source                                                                                 |
| ------------------- | ------------------ | -------------------------------------------------------------------------------------- |
| `global.button.css` | `.button` or `.bt` | [Github](https://github.com/minimaldesign/mCSS/blob/main/src/styles/global.button.css) |

The `button` component is one of the global components provided by mCSS. You can use `.button` or its shorthand `.bt` on `<button>`, `<input>` and `<a>` HTML elements. They will all render the same.

You can combine modifiers however you want (see exmamples below) but note that the `.bt-outline` modifier will only work on buttons that have a solid fill color to start.

| Class | Modifier                  | Example                                                              |
| ----- | ------------------------- | -------------------------------------------------------------------- |
| `.bt` |                           | <button class="bt">Default Button</button>                           |
| `.bt` | `.bt-md`                  | <button class="bt bt-md">Medium Size</button>                        |
| `.bt` | `.bt-lg`                  | <button class="bt bt-lg">Large Size</button>                         |
| `.bt` | `.bt-primary`             | <button class="bt bt-primary">Primary CTA</button>                   |
| `.bt` | `.bt-primary .bt-outline` | <button class="bt bt-primary bt-outline">Outlined CTA</button>       |
| `.bt` | `.bt-yes`                 | <button class="bt bt-yes">Outlined CTA</button>                      |
| `.bt` | `.bt-maybe`               | <button class="bt bt-maybe">Outlined CTA</button>                    |
| `.bt` | `.bt-no`                  | <button class="bt bt-no">Outlined CTA</button>                       |
| `.bt` | SVG inline inside button  | <button class="bt"><Fragment set:html={arrow} />Icon Button</button> |
| `.bt` | `disabled` attribute      | <button class="bt" disabled>Disabled Button</button>                 |

</section>
